Get ComplexTypeProperty

概要

既存のComplexTypeProperty情報を取得する

必要な権限

read

制限事項

·           リクエストヘッダーのContent-Typeは全てapplication/jsonとして扱う

·           リクエストボディはjson形式のみ受け付ける

·           レスポンスヘッダーのContent-Typeはapplication/jsonのみをサポートし、レスポンスボディはjson形式とする

·           $formatクエリオプションにatom または xmlを指定した場合、エラーとはならないが、レスポンスボディのデータの保証はない

リクエスト

リクエストURL

/{Cell_name}/{Box_name}/{odata_colleciton_path}/$metadata/ComplexTypeProperty(Name='complextype_property_name',_ComplexType.Name='ComplexType_name')
