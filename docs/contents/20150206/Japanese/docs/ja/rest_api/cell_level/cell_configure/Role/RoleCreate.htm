Role登録API

概要

Roleを登録する  

必要な権限

auth

制限事項

·           リクエストヘッダーのAcceptは無視される

·           リクエストヘッダーのContent-Typeは全てapplication/jsonとして扱う

·           リクエストボディはjson形式のみ受け付ける

·           レスポンスヘッダーのContent-Typeはapplication/jsonのみをサポートし、レスポンスボディはjson形式とする

·           $formatクエリオプションは無視される

リクエストURL

/{Cell_name}/__ctl/Role

リクエストボディ

項目名

概要

有効値

必須

備考

Name

Role名

桁数：1～128
文字種:半角英数字と-(半角ハイフン)と_(半角アンダーバー)
ただし、先頭文字に-(半角ハイフン)と_(半角アンダーバー)は指定不可

○

 

_Box.Name

関係対象のBox名

桁数：1～128
文字種：半角英数字と-(半角ハイフン)と_(半角アンダーバー)
ただし、先頭文字に-(半角ハイフン)と_(半角アンダーバー)は指定不可
説明：Box登録APIにて登録済みのBoxのNameを指定
特定のBoxと関連付けない場合はnullを指定

×
