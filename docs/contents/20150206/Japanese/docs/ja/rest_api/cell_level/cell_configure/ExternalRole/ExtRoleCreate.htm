ExtRole登録API

概要

ExtRoleを登録する

制限事項

なし

OData 制限

·        リクエストヘッダーのAcceptは無視される

·        リクエストヘッダーのContent-Typeは全てapplication/jsonとして扱う

·        リクエストボディはjson形式のみ受け付ける

·        レスポンスヘッダーのContent-Typeはapplication/jsonのみをサポートし、レスポンスボディはjson形式とする

·        $formatクエリオプションは無視される

必要な権限

auth

備考

ExtRole

外部ロール名(URL)

桁数：1～1024
URIの形式に従う
scheme：http / https / urn
Boxに紐付いている場合:/{アプリセル名}/__role/__/{Role名}
※ただし、BoxにSchema情報が登録されていない場合、Boxに紐付いていないとみなされる
Boxに紐付いていない場合:/{Cell名]/__role/__/{Role名}

○

 

_Relation.Name

関係対象のRelation名

桁数：1～128
文字種:半角英数字と-(半角ハイフン)と_(半角アンダーバー)
と+(プラス)と:(コロン)
ただし、先頭文字に_(半角アンダーバー):(コロン)は指定不可
Relation登録APIにて登録済みのRelation
null

○

 

_Relation._Box.Name

関係対象のRelationが紐づくBox名

桁数：1～128
文字種:半角英数字と-(半角ハイフン)と_(半角アンダーバー)
Relation登録APIにて登録済みのRelationが紐づくBox名
null
