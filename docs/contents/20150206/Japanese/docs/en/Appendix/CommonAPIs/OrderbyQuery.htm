$orderby Query

概要

This query allows you to sort the collection of records. By default, the order is ascending. Use desc to reverse the order.

If you specify the name of the property that does not exist, the $Orderby query ignores it.

If you specify the property name of the array type, the $Orderby returns the 400 error.

リクエストクエリ

$orderby={sortKey} {option}, · · ·

{sortkey} = {propertyName} {
{option} = you can specify multiple options by using comma(s)

 

Path

概要

{PropertyName}

Property name to specify the sort key

{Option}

Ascending: asc, which is a default value

Descending order: desc

CURLサンプル

1.     To get a list that is sorted in descending order by the Test Properties

$orderby=Test%20desc

2.     To get a list that is sorted in ascending order to a new list Test sorted in descending order

$orderby=ID%20asc,Test%20desc
