Property $links削除

概要

 $ linkのPropertyを削除する

必要な権限

なし

制限事項

·           リクエストヘッダーのContent-Typeは全てapplication/jsonとして扱う

·           リクエストボディはjson形式のみ受け付ける

·           レスポンスヘッダーのContent-Typeはapplication/jsonのみをサポートし、レスポンスボディはjson形式とする

·           $formatクエリオプションにatom または xmlを指定した場合、エラーとはならないが、レスポンスボディのデータの保証はない

リクエストURL

EntityTypeとLinkしている場合

/{Cell_name}/{Box_name}/{Collection_name}/$metadata/Property(Name='{Property_name}',_EntityType.Name='{EntityType_name}')/$links/_EntityType('{EntityType_name}')
