オープンソースプロジェクトPersoniumでは様々な形での参加・コントリビューションを歓迎しております。

## 質問

Personiumのコンセプトや使い方を質問する場合は[Discordコミュニティー](https://discord.gg/RgwCgvc3Ur)でお願いします。  


各チャンネルの使い方は以下のようになっています。

|チャンネル名|話題|
|----------|----|
|#general|一般的な内容|
|#random|雑談|
|#infra|Personiumサーバ構築|
|#appdev|Personiumを使ったアプリ開発|
|#dev|Personium自体の開発|

## ドキュメントの不備指摘・改善
